## 📖 About This Repository

Welcome to my **Python Mastery Journey** — a structured, self-paced learning repository where I document every step of my growth as a Python developer. This repo is not just a collection of scripts; it's a living record of my progression from writing my first `print("Hello, World!")` all the way to building advanced, real-world Python applications.

Every concept I learn is broken into **three difficulty tiers** — Easy, Medium, and Hard — so that each topic is explored with increasing depth and complexity. Whether you're a fellow learner, a mentor, or just a curious visitor, feel free to explore, fork, and follow along.

## 🎯 Difficulty Levels Explained

| Level | Badge | Description | Goal |
|-------|-------|-------------|------|
| **Easy** | 🟢 | Core concepts, syntax-focused exercises | Build foundational confidence |
| **Medium** | 🟡 | Combined concepts, mini-projects, real-world scenarios | Develop problem-solving instincts |
| **Hard** | 🔴 | Algorithmic challenges, edge cases, performance-aware code | Think like an engineer |

Each difficulty level is carefully designed to build upon the previous one. Skipping directly to **Hard** without completing **Easy** and **Medium** is like skipping leg day — technically possible, but you'll regret it. 😅

## 💡 Why Three Modes?

Most tutorials teach you *what* Python can do. This repo teaches you *how well* you actually understand it. By solving the same concept at three different difficulty levels, I force myself to:

- **Think critically** — not just copy-paste syntax
- **Identify gaps** — if I can't solve the hard version, I revisit the concept
- **Celebrate growth** — looking back at Easy problems after mastering Hard ones is incredibly motivating
